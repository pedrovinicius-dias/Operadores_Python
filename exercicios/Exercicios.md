## Operadores Aritméticos

**1.** Qual o resultado da expressão abaixo? Justifique a ordem das operações.
```
7 + 3 * 2
```

**2.** Qual o resultado de `10 % 3`? O que o operador `%` representa?

**3.** Escreva a expressão em Python que calcula a área de um quadrado de lado 4.

**4.** Qual o resultado de `2 ** 3`? O que o operador `**` representa?

---

## Operadores Relacionais

**5.** As expressões abaixo são verdadeiras ou falsas? Justifique.
- `10 > 7`
- `4 == 4.0`
- `15 != 15`

**6.** Escreva a expressão em Python que verifica se a nota de um aluno é suficiente para aprovação, considerando nota mínima igual a 6.

**7.** Dado o código abaixo, qual será a saída no terminal?
```python
x = 8
print(x >= 10)
print(x != 8)
print(x < 10)
```

## Operadores Lógicos

**8.** As expressões abaixo são verdadeiras ou falsas? Justifique.
- `(3 > 2) and (5 > 10)`
- `(3 > 2) or (5 > 10)`
- `not (3 > 2)`

**9.** Escreva a condição lógica em Python que verifica se uma pessoa pode tirar a carteira de motorista, considerando que ela deve ter **no mínimo 18 anos** e **ter feito o exame**.

**10.** Dado o código abaixo, qual será a saída no terminal?
```python
a = True
b = False

print(a and b)
print(a or b)
print(not a)
```
