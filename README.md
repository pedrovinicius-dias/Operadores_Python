# Operadores 🧮

### Projeto de Extensão — BCC 2026.1 | Lógica de Programação

---

## Sobre o Projeto

Este repositório reúne todo o material produzido pelo Grupo 2 no Projeto de Extensão da disciplina de Lógica de Programação, orientado pelo Prof. Pedro Lucas Luz.

O nosso tema foi Operadores — aqueles símbolos que fazem o código sair do campo das ideias e começar a calcular, comparar e tomar decisões de verdade. Sem eles, variáveis são só dados tristes e isolados na memória RAM.

Aqui você vai encontrar pesquisas, exemplos práticos em Python, exercícios resolvidos, o slide da apresentação, o roteiro completo da videoaula e o relatório final de extensão. Tudo organizado e pensado para ser didático, direto e (na medida do possível) agradável de consumir.

---


## Conteúdos Abordados

- **Operadores Aritméticos** — `+`, `-`, `*`, `/`, `%`, `**` e a boa e velha precedência de operações.
- **Operadores Relacionais** — `==`, `!=`, `>`, `<`, `>=`, `<=` e o perigo clássico de confundir `=` com `==`.
- **Operadores Lógicos** — `and`, `or`, `not` e como eles combinam condições para tomar decisões complexas.
- **Aplicações Práticas** — Como os três tipos trabalham juntos em cenários reais de programação.
